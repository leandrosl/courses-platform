# Courses Platform

Um projeto de uma plataforma de cursos, onde estudantes e professores poderão ter acesso. Este é um side project com o objetivo de aprimorar práticas de desenvolvimento utilizando agentes de IA.

## Módulos principais
- autenticação
- catálogo de cursos
- trilhas de aprendizagem
- inscrição e acesso
- progresso do aluno
- avaliações e quizzes
- comentários e discussões
- certificados
- notificações
- dashboard administrativo

## Tecnologias utilizadas
- C Sharp
- Javascript
- React
- Flutter
- Kotlin
