# Courses Platform

Um projeto para aprimorar técnicas de desenvolvimento de sistemas utilizando agentes de IA

## Estrutura
```text
/courses-platform
  /backend-csharp
  /backend-javascript
  /frontend-react
  /mobile-flutter
  /mobile-android
```

## Módulos Principais
- autenticação
- catálogo de cursos
- trilhas de aprendizagem
- inscrição e acesso
- progresso do aluno
- avaliações e quizzes
- comentários e discussões
- certificados
- notificações
- dashboard administrativo

## Funcionalidades
1. Autenticação e acesso
- cadastro
- login
- refresh token
- recuperação de senha
- perfis
- acesso a cursos inscritos
- controle de áreas administrativas

2. Catálogo de cursos
- listagem de cursos
- busca
- filtros por categoria
- cursos em destaque
- página de detalhe
- autor/instrutor
- carga horária
- nível
- tags

3. Estrutura pedagógica
- cursos
- módulos
- aulas
- trilhas
- pré-requisitos
- materiais complementares

4. Inscrição e acesso ao curso
- inscrição em curso
- liberação de acesso
- cursos concluídos
- bloqueio por pré-requisito
- progresso geral

5. Progresso do aluno
- aula iniciada
- aula concluída
- porcentagem de progresso
- última aula vista
- continuar de onde parou
- progresso por módulo
- progresso por trilha

6. Quizzes e avaliações
- perguntas objetivas
- tentativas
- nota mínima
- feedback
- status de aprovação
- bloqueio/liberação de certificado

7. Certificados
- emissão após conclusão
- código de validação
- exportação PDF
- verificação pública

8. Comentários e discussões
- comentários por aula
- respostas
- moderação
- denúncia
- destaque de resposta do instrutor

9. Notificações
- novo curso
- nova aula liberada
- curso concluído
- aprovação em quiz
- lembretes de continuidade

10. Painel administrativo
- gestão de cursos
- gestão de módulos e aulas
- publicação
- gestão de usuários
- métricas de inscrição
- métricas de conclusão
- desempenho por curso

## Regras
- Sempre utilizar boas práticas de desenvolvimento
- Tentar ao máximo deixar o código mais simples possível, para fácil entendimento humano
