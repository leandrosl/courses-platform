# Front React com Docker

Aplicação front-end em React com Vite, escrita em JavaScript, pronta para desenvolvimento via Docker.

## Stack

- React 19
- Vite 7
- Docker Compose
- JavaScript puro, sem TypeScript

## Como subir

```bash
cd front-react
docker compose up --build
```

A aplicação ficará disponível em `http://localhost:5173`.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
